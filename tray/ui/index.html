Text history

Everything spoken is logged as text to ~/.claude/vox/history.jsonl so the History tab can browse and replay it. Entries are deleted after the retention window below. The single most-recent readout is always kept separately, so "vox, repeat that" works even with history off.

minutes (0 = keep, capped at 500 entries)

Markdown handling

Formatting is spoken as pauses, never syntax: bold and code markers vanish, headers and bullets become their own sentences, code blocks say "code block omitted", links read their text. This is a deterministic script — not a prompt, no AI involved — so it's instant and runs on everything spoken. Edits apply from the next utterance.

Audio files

⚠️ Saved audio is uncompressed WAV — roughly 3 MB per spoken minute. Leaving this on without a delete-after window can use a lot of disk. Off by default; applies to the Claude hook and this app (the vox CLI's own saving is configured in vox itself).
